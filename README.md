## Installation

Require [PHPUnit](https://phpunit.de/index.html).

```bash
composer require --dev phpunit/phpunit ^10
```

**Running tests**

```bash
vendor/bin/phpunit test/RTests.php
vendor/bin/phpunit test/RDBTests.php
```
